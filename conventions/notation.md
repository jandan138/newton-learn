# Notation

| 记号 | 含义 | 约定 |
| --- | --- | --- |
| `x_w` | 世界坐标中的位置 | 下标 `_w` 表示 world frame |
| `x_b` | 刚体局部坐标中的位置 | 下标 `_b` 表示 body/local frame |
| `R` | 旋转矩阵 | 默认从局部到世界 |
| `q` | 广义坐标 | 平移单位 `[m]`，旋转单位 `[rad]` |
| `qdot` | 广义速度 | 文中写作 `q̇`，代码里写 `qdot` |
| `qddot` | 广义加速度 | 文中写作 `q̈` |
| `M` | 质量矩阵 | 默认 SPD，维度随自由度变化 |
| `J` | 约束或关节 Jacobian | 默认行对应约束、列对应自由度 |
| `lambda` | 约束乘子 / 接触冲量 | 代码里写 `lambda` 或 `lmbda` |
| `tau` | 主动广义力 | 单位 `[N or N·m]` |
| `h` | 时间步长 | 与 `dt` 同义时优先写 `h` |
| `x_n` / `x_{n+1}` | 离散时间状态 | 子步用上标 `x^{(k)}` |
| `phi` | 穿透 / 距离函数 | 接触中默认 `< 0` 表示穿透 |
| `n` | 接触法向 | 默认从 A 指向 B |
| `SDF` | signed distance field | 距离为正表示在外部 |

## 坐标与旋转

- 四元数默认写成 `(x, y, z, w)`，除非上游源码明确相反。
- 列向量默认；矩阵右乘时在文中明确写出。
- Newton 当前默认 builder up-axis 记为 `Z-up`，除非某个资产导入流程明确改写。
- 若外部资产是 `Y-up`，在章节里写清从资产坐标到 Newton builder 坐标的变换。

## MPM / FEM 额外约定

- 粒子索引用 `p`，网格节点索引用 `g`。
- 形变梯度写 `F`，应力写 `P` 或 `sigma`，不混用。
- `P2G` / `G2P` 保留英文缩写，不翻译。
