# Glossary

> 按英文术语首字母排序。

| 术语 | 英文 | 简释 |
| --- | --- | --- |
| ABA | articulated body algorithm | 关节系统前向动力学算法 |
| 伴随法 | adjoint method | 通过反向系统传播灵敏度 |
| APIC | affine particle-in-cell | 保角动量的粒子-网格传输 |
| 刚体链 | articulation | 由关节连接的多刚体系统 |
| 宽相碰撞 | broadphase | 候选碰撞对筛选阶段 |
| CRBA | composite rigid body algorithm | 组装质量矩阵的算法 |
| 共轭梯度 | conjugate gradient | 稀疏 SPD 线性系统迭代法 |
| CUDA graph | CUDA graph | 复用 launch 图以降低调度开销 |
| Delassus 算子 | Delassus operator | 约束空间中的有效惯量算子 |
| EPA | expanding polytope algorithm | GJK 相交后的穿透深度求解 |
| XPBD | extended position-based dynamics | 带柔顺性的 PBD 扩展 |
| 广义坐标 | generalized coordinates | 关节空间的位置变量 |
| 广义力 | generalized force | 关节空间中的驱动力 / 力矩 |
| 广义速度 | generalized velocity | 关节空间的速度变量 |
| GJK | Gilbert-Johnson-Keerthi | 凸体最近距离 / 相交检测算法 |
| hydroelastic | hydroelastic contact | 连续分布式接触模型 |
| 隐函数定理 | implicit function theorem | 隐式求解器反向传播基础 |
| 关节雅可比 | Jacobian | 把广义速度映到约束或任务空间 |
| 互补问题 | LCP / NCP | 约束与接触常见数学形式 |
| 质量矩阵 | mass matrix | 刚体系统惯性的全局表示 |
| MPM | material point method | 粒子-网格混合连续体方法 |
| MPR | Minkowski portal refinement | 凸体接触查询算法 |
| 窄相碰撞 | narrowphase | 精细接触检测阶段 |
| OpenUSD | OpenUSD | 通用场景描述与资产交换格式 |
| 近端 ADMM | proximal ADMM | 带近端项的交替方向乘子法 |
| RNEA | recursive Newton-Euler algorithm | 反向动力学经典算法 |
| 随机平滑 | randomized smoothing | 缓和不可微接触的技巧 |
| 反向模式自动微分 | reverse-mode AD | 从标量损失反传梯度 |
| 有符号距离场 | signed distance field | 几何到边界的带符号距离 |
| 软约束 | soft constraint | 允许少量违反的惩罚式约束 |
| 空间向量 | spatial vector | 组合线速度/角速度或力/力矩的 6D 量 |
| tile | tile | Warp 中块级局部数据组织方式 |
| VBD | variational backward dynamics | 变分后向动力学框架 |
| Warp kernel | Warp kernel | Warp 中的 GPU 并行计算单元 |
